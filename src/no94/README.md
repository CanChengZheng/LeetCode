# 题意

提供一个二叉树，返回二叉树中序遍历的节点值。

举例：   
二叉树：`[1,null,2,3]`
```
   1
    \
     2
    /
   3
```
返回`[1,3,2]`


# 思路

## 思路1

实现使用递归算法。

## 思路2

使用栈进行实现。[参考链接 #2 Iterating method using Stack](https://leetcode.com/problems/binary-tree-inorder-traversal/solution/#approach-2-iterating-method-using-stack-accepted)
