# 题意

给定一个二叉树，找到一条路径，路径上节点值的总和是所有路径中最大的。

针对这个问题，一条路径是指从任意的节点起始，通过节点间的联系，到某个节点结束。这条路径必须至少包含一个节点，并且可以不经过根节点。

举例：
提供以下的字符串：
```
       1
      / \
     2   3
```
返回 `6`.


# 思路

首先因为是二叉树，我们可以考虑使用递归来解决问题。那么使用递归算法，就相当于将一个大问题划分为相同的小问题进行解决。
假设现在是根节点`root`，那么当前节点的`路径最大值`由`自身的值`以及`左右两棵子树的路径最大和`决定。F(root) = val + F(left) + F(right)。当左右节点的路径最大和如果是负数，则忽略。
需要注意到的一个情况是，求当前节点的路径最大值上述所说。而当前节点返回给父节点的路径最大和是由`自身的值`以及左右两棵子树的`两个路径最大和中的较大数`组成的。
